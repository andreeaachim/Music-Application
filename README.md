# **Music Application**

## _Introduction_
This application is a homework for the SEF laboratory. 
Jira tracking system on the [Jira example](https://musicapplication.atlassian.net/jira/software/projects/MAS/boards/2) and [Github example](https://github.com/eliadaa/Music-App.git)

## _Usage_
This application is used for listening, selecting and adding new/existing music in playlists.
There are 2 scenarios: 
You can login as a client, a user. Being able to add music to playlists, like artists, like their music, and get notified when those artists anounce events.
Or you can login as an artist being able to add new music for users to listen and get notifications when users like your music.
